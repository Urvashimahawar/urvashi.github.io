Data Management

Import product data from CSV files or load directly from GitHub.

To load images from GitHub:

Your GitHub repository should have an images/ folder containing product images and CSV files.

CSV file image URLs should be in this format:

  • Full GitHub URL: https://raw.githubusercontent.com/Urvashimahawar/urvashi.github.io/main/images/earrings1.jpg
  • Relative path: earrings1.jpg (just file name)

CSV Format:

Your CSV file should have the following columns (in this order):

productId, productName, category, quantity, price, imageUrl1, imageUrl2, imageUrl3

Example:

P001,Beautiful Earrings Size-1,Jewelry,50,599.99,earrings1.jpg,earrings2.jpg,earrings3.jpg
P002,Handmade Decor,Home Decor,30,149.50,decor1.jpg,decor2.jpg,decor3.jpg
